Key Design Features:

✓ 6 top-level items (streamlined, focused)
✓ "Search..." prompt for intuitive filtering
✓ Screenshot at top level for quick access
✓ Logical grouping of related functions
✓ 2 levels deep maximum (fast navigation)
✓ Consistent "Back" navigation throughout
✓ All menus positioned based on context
✓ Auto-sizing with --lines (no scrolling, no dead space)

═══════════════════════════════════════════════════════════════════

Menu Category Philosophy:

󰏘 Appearance  → How things LOOK
󰒓 System      → How things WORK  
󰏖 Install     → What you ADD
󰄀 Screenshot   → Quick capture tool
󰀉 Preferences → How you CONFIGURE
󰐥 Power       → How you EXIT

═══════════════════════════════════════════════════════════════════

Design Decisions:

✗ Removed Launch menu - Apps launched via wofi (Super+Space)
✗ Removed Actions menu - Items moved to top level or other menus
✓ Screenshot at top level - Frequently used, deserves prominence
✓ Streamlined Preferences - Default apps, keybindings, autotiling, config
✓ Install menu ready for package manager expansion
✓ "Search..." prompt instead of "BunkerOS" - more intuitive
✓ Default Apps - Easy customization of mod+e, mod+b, mod+f, etc.
