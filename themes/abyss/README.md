# Abyss Theme

**Deep Ocean Underwater Mission**

A tactical dark theme inspired by deep ocean environments and underwater operations, featuring deep navy blues, luminous sky blue accents, and vintage lume colors for a focused, immersive workspace.

## Inspiration

The Abyss theme draws inspiration from:
- Deep ocean underwater environments
- Submarine and tactical diving operations
- Bioluminescent deep-sea creatures
- The mysterious depths of the ocean abyss
- Vintage dive watch aesthetics with lume accents

This creates a dark, focused workspace with calming ocean blues and luminous accents, perfect for deep concentration and tactical operations.

## Color Palette

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Sky Blue** | `#7BA5D4` | Focused elements, important accents |
| **Deep Navy** | `#1E3A5F` | Prominent elements |
| **Deep Ocean Black** | `#0A1628` | Main background |
| **Dark Navy Blue** | `#152844` | Secondary background |
| **Cream Lume** | `#E8E6DC` | Primary text |
| **Muted Navy** | `#1A2F4A` | Borders, inactive elements |
| **Medium Blue** | `#2E4A6B` | Icons, secondary text |
| **Gold** | `#D4A574` | Alerts, important items |

## Theme Characteristics

- **Temperature**: Cool
- **Style**: Tactical/Underwater
- **Best For**: Deep work, focused sessions, low-light environments
- **Contrast**: High (light text on very dark backgrounds)
- **Unique Feature**: Deep ocean immersion with luminous accents

## Notes

The Abyss theme provides a deeply immersive environment perfect for tactical operations and focused work. The combination of near-black backgrounds with luminous blue accents mimics the experience of a deep underwater mission, where every element glows with purpose.
